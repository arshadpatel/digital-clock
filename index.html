Description

This is an Application made using Java Swings. Java Swing is a part of Java Foundation Classes (JFC) that is used to create window-based applications. It is built on the top of AWT (Abstract Windowing Toolkit) API and entirely written in java.

Unlike AWT, Java Swing provides platform-independent and lightweight components.

The javax.swing package provides classes for java swing API such as JButton, JTextField, JTextArea, JRadioButton, JCheckbox, JMenu, JColorChooser etc.

This is a Digital Clock application which not only shows time but also has features such as Stopwatch and Timer in which all uses Digitial-7 Font. It is a very light weight application made using only single frame, it doesnot create new frame each time we click the buttons. It runs on single frame from the time we run it to the time we close it.

Clock

When we run the application it displays the current time which has hours:minutes:seconds and also includes current date and day. This frame consists of two buttons Stopwatch and Timer. On clicking StopWatch the application displays stopwatch and on clicking timer the application displays timer.

Stopwatch

On clicking StopWatch Button Stopwatch is displayed which consist of start button on clicking this button the time count will begin from zero, if required you can also pause the watch on clicking pause button. It also consists of Reset Button which resets it to 00:00:00 It is similar to the stopwatch which claculates the amount of time that elapses between its activation and deactivation.

Timer

On clicking Timer button Timer is dispalyed where you are provided with a spinner using which you can set the time and on clicking start the timer will begin. You can also pause the timer using pause time the timer will stop. to resume the timer click on the resume button. To reset or delete the current timer click on the delete button it will reset the timer.
